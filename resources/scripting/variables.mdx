---
title: "Variables"
description: "Global variables available in adore embed and message templates."
icon: "signature"
---

These variables work anywhere adore supports embed code or custom messages - for example `welcome`, `leave`, `boost`, and vanity claim notifications.

<Note>
  Command-specific variables (bump reminders, level-up messages, punishment DMs, Roblox, YouTube, etc.) are handled by those commands directly.
</Note>

<Tabs>
  <Tab title="Guild">

    | `{guild.id}`                      | The guild's ID                                                       |

    | `{guild.created_at_timestamp}`    | The guild's creation date in UNIX                                    |
    | `{guild.emoji_count}`             | The guild's emoji count                                              |
    | `{guild.role_count}`              | The guild's role count                                               |

  <Tab title="User">

    | `{user.name}`                  | The user's name                                        |

    | `{user.guild_avatar}`          | The user's guild profile picture                       |
    | `{user.display_avatar}`        | The user's available profile picture                   |
    | `{user.joined_at}`             | The user's guild join date in UTC                      |
    | `{user.joined_at_timestamp}`   | The user's guild join date in UNIX                     |

    | `{user.created_at_timestamp}`  | The user's account creation date in UNIX               |
    | `{user.display_name}`          | The user's current display name                        |
    | `{user.nick}`                  | The user's server display name                         |
    | `{user.nickname}`              | Alias for `{user.nick}`                                |

    | `{user.boost_since}`           | The user's initial guild boost date                    |
    | `{user.boost_since_timestamp}` | The user's initial guild boost date in UNIX            |
    | `{user.color}`                 | The user's top role hex code                           |
    | `{user.top_role}`              | The user's top role name, defaults to N/A              |

    | `{user.bot}`                   | `Yes`/`No` if the user is a bot                        |
    | `{user.badges_icons}`          | The user's profile badges with emotes (possibly `N/A`) |
    | `{user.badges}`                | The user's profile badges in text (possibly `N/A`)     |
    | `{user.join_position}`         | The user's join position                               |
    | `{user.join_position_suffix}`  | The user's join position with suffix (e.g. `42nd`)     |

  <Tab title="Last.fm">
    <Info>
      Only available in `,lastfm mode` and custom now-playing templates. Do **not** use `{user.plays}`, `{user.url}`, or `{author}` - those are not valid.
    </Info>

    | Variable | Output |
    | -------- | ------ |
    | `{lastfm.name}` | Last.fm username |
    | `{lastfm.avatar}` | Last.fm profile picture |
    | `{lastfm.plays}` | Total scrobbles |
    | `{proper(lastfm.plays)}` | Total scrobbles (formatted) |
    | `{lastfm.artist_crown}` | Crown emoji if you lead the artist in this server |
    | `{lastfm.url}` | Last.fm profile URL |
    | `{track.name}` | Current track title |
    | `{lower(track.name)}` | Track title in lowercase |
    | `{track.url}` | Last.fm track URL |
    | `{track.plays}` | Your plays for the track |
    | `{proper(track.plays)}` | Track plays (formatted) |
    | `{track.spotify_url}` | Spotify track URL |
    | `{track.release_date}` | Release date (Spotify) |
    | `{track.duration}` | Duration (Spotify) |
    | `{artist.name}` | Artist name |
    | `{lower(artist.name)}` | Artist name in lowercase |
    | `{artist.plays}` | Your plays for the artist |
    | `{proper(artist.plays)}` | Artist plays (formatted) |
    | `{artist.url}` | Last.fm artist URL |
    | `{artist.image}` | Artist image (Spotify) |
    | `{album.name}` | Album title |
    | `{lower(album.name)}` | Album title in lowercase |

    | `{album.cover}` | Album cover |
    | `{user.display_name}` | Discord display name |
    | `{user.name}` | Discord username |
    | `{user.mention}` | Discord @mention |
    | `{user.avatar}` | Discord avatar URL |
