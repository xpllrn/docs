Donator perks are rewards given to [Personal Premium](/overview/premium) subscribers. They are tied to your **account**, not a single server.

## How to obtain Donator Perks

Personal Premium can be purchased by opening a ticket in our [Support Server](https://discord.gg/q3e53k4qKQ).

<Info>
  **Price:** \$5/mo · **Payment methods:** PayPal · Bank Transfer · Bitcoin / Ethereum / Litecoin

  See [Premium Plans](/overview/premium) and live pricing at [adore.rest/premium](https://adore.rest/premium).
</Info>

## Grok AI

Chat with Grok-powered AI.

<CodeGroup>

```javascript Syntax
,grok (question)
```

```javascript Example
,grok explain how Discord bots work
```

</CodeGroup>

## Last.fm Mode

Customize the embed layout of the `nowplaying` command.

<CodeGroup>

```javascript Syntax
,lastfm mode (embed script)
,lastfm mode (check|reset)
```

```javascript Example
,lastfm mode {embed}$v{title: Now Playing}$v{description: {track.name}}$v{thumbnail: {album.cover}}
```

</CodeGroup>

See the full variable list on the [Last.fm integration page](/integrations/lastfm#custom-now-playing-embed-personal-premium).

## Last.fm Custom Reactions

Customize the reactions that appear under the `nowplaying` command.

<CodeGroup>

```javascript Syntax
,lastfm reaction (upvote emoji) (downvote emoji)
```

```javascript Example
,lastfm reaction 👍 👎
```

</CodeGroup>

## Last.fm Custom Command

Set a custom trigger message that shows your now playing.

<CodeGroup>

```javascript Syntax
,lastfm cc (message)
```

```javascript Example
,lastfm cc .
```

</CodeGroup>
