## What is adore?

adore is a free, multipurpose Discord bot for moderation, security, economy, leveling, tickets, social integrations, and AI. Everything your server will adore — in one bot.

### Quick start

1. [Invite adore](https://adore.rest) to your server with the **Administrator** permission (recommended for full functionality).
2. Run `,help` to browse commands, visit [adore.rest/commands](https://adore.rest/commands), or read the [wiki](https://wiki.adore.rest).
3. Set up [anti-nuke protection](/security/antinuke) and [fake permissions](/security/fake-permissions) before granting staff access.
4. Customize your server with `,prefix`, `,welcome`, `,autorole`, and more.
