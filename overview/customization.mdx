## Overview

Personalize adore's appearance in your server by setting a custom avatar, banner, bio, nickname, or embed color. Requires **Server Premium**.

<Warning>
  This feature is actively monitored and logged. Uploading inappropriate content
  will lead to **immediate removal** of the bot from your server and **forfeiture
  of access**.
</Warning>

<Callout icon="key" color="#558184" iconType="regular">
  Only the **server owner** can use these commands. Requires **Server Premium**.
</Callout>

## Customizing adore's avatar

Change adore's avatar in your server with the `customize avatar` command.

<CodeGroup>

```javascript Syntax
,customize avatar (url)
,customize avatar remove
```

```javascript Example
,customize avatar https://example.com/avatar.png
```

</CodeGroup>
<br />

## Customizing adore's banner

<CodeGroup>

```javascript Syntax
,customize banner (url)
,customize banner remove
```

```javascript Example
,customize banner https://example.com/banner.png
```

</CodeGroup>
<br />

## Customizing adore's bio

<CodeGroup>

```javascript Syntax
,customize bio (text)
,customize bio remove
```

```javascript Example
,customize bio Your server bio here
```

</CodeGroup>
<br />

## Other customization

| Command | Description |
| ------- | ----------- |
| `,prefix (symbol)` | Set the server command prefix |
| `,selfprefix (symbol)` | Set your personal prefix (Personal Premium) |
| `,customize name (name)` | Change adore's nickname in the server |
| `,customize color (hex)` | Set the default embed color |
