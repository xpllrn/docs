## Overview

Personalize adore's appearance in your server by setting a custom avatar, banner, and/or bio. This is a **Premium** feature that makes adore feel like your own bot.

<Warning>
  This feature is actively monitored and logged. Uploading inappropriate content
  will lead to **immediate removal** of the bot from your server and **forfeiture
  of access**.
</Warning>

<Callout icon="key" color="#ff5d8f" iconType="regular">
  Only the **server owner** can use these commands. Requires **server Premium**.
</Callout>

## Customizing adore's avatar

Change adore's avatar in your server with the `customize avatar` command.

<CodeGroup>

```javascript Syntax
,customize avatar (image URL)
```

```javascript Example
,customize avatar https://adore.rest/logo.png
```

</CodeGroup>

## Customizing adore's banner

Change adore's banner with the `customize banner` command.

<CodeGroup>

```javascript Syntax
,customize banner (image URL)
```

```javascript Example
,customize banner https://adore.rest/banner.png
```

</CodeGroup>
<br />

## Customizing adore's bio

Set adore's bio with the `customize bio` command.

<Info>
    If you do not manually include **https://adore.rest** or **discord.gg/2JYMZWeu9s** in the bio, it will be automatically added at the bottom.
</Info>

<CodeGroup>

```javascript Syntax
,customize bio (text)
```

```javascript Example
,customize bio Everything your server will adore.
```

</CodeGroup>
<br />

## Other customization

| Command | Description |
| ------- | ----------- |
| `,prefix (symbol)` | Set the server command prefix |
| `,selfprefix (symbol)` | Set your personal prefix (donator perk) |
| `,customize name (name)` | Change adore's nickname in the server |
| `,customize color (hex)` | Set the default embed color |
