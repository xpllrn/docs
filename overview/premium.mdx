## Plans

adore is **free** for the full moderation, security, economy, leveling, and integration toolkit. Premium and donator tiers unlock extras.

| Tier | Price | What you get |
| ---- | ----- | ------------ |
| **Free** | $0 | Full moderation, security, economy, leveling, tickets, integrations |
| **Premium** | $4.99/mo | Per-server bot customization, priority AI, higher limits |
| **Donator** | Support server | Personal perks like self-prefix, grok, transcribe, transparent |

<Info>
  Upgrade or purchase perks in our [Support Server](https://discord.gg/q3e53k4qKQ).
</Info>

## Donator perks

Donator perks are personal rewards for supporting adore. They apply across all servers you use adore in.

<AccordionGroup>
  <Accordion title="Self prefix">
    Set your own command prefix across all servers.
    <CodeGroup>
    ```javascript Syntax
    ,selfprefix (prefix)
    ```
    ```javascript Example
    ,selfprefix !
    ```
    </CodeGroup>
  </Accordion>
  <Accordion title="Self aliases">
    Create personal command aliases that work in any server.
    <CodeGroup>
    ```javascript Syntax
    ,selfalias add (alias) (command)
    ```
    ```javascript Example
    ,selfalias add b balance
    ```
    </CodeGroup>
  </Accordion>
  <Accordion title="Grok AI">
    Chat with Grok-powered AI via `,grok`.
    <CodeGroup>
    ```javascript Syntax
    ,grok (question)
    ```
    ```javascript Example
    ,grok explain quantum computing simply
    ```
    </CodeGroup>
  </Accordion>
  <Accordion title="Voice transcription">
    Transcribe voice messages with `,transcribe`.
    <CodeGroup>
    ```javascript Syntax
    ,transcribe (reply to voice message)
    ```
    </CodeGroup>
  </Accordion>
  <Accordion title="Background removal">
    Remove the background from an image.
    <CodeGroup>
    ```javascript Syntax
    ,transparent (image URL)
    ```
    ```javascript Example
    ,transparent https://adore.rest/example.png
    ```
    </CodeGroup>
  </Accordion>
</AccordionGroup>
