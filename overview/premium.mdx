## How to purchase

<Steps>
  <Step title="Join the support server">

  </Step>
  <Step title="Choose a plan">
    Pick **Personal Premium**, **Server Premium**, or a **Private Instance** for your server or account.

    Pay via **PayPal**, **Bank Transfer**, or **Bitcoin / Ethereum / Litecoin**. Our team will activate your subscription or instance.

## Personal Premium {#personal-premium}

Perks that follow you everywhere - tied to your **account**, not a single guild.

| | |
| --- | --- |
| **Price** | $5/mo |

<AccordionGroup>
  <Accordion title="What's included" defaultOpen>
    - **3× rate limits** (15/5s)
    - **1.5× economy multiplier**
    - **Personal prefix** - `,selfprefix`
    - **Personal aliases** - `,selfalias`
    - **Grok AI** - `,grok`
    - **Voice transcription** - `,transcribe`
    - **Background removal** - `,transparent`
    - **Instagram command**
    - **Selfpurge command**
    - **Custom AFK embeds**
    - **Extra giveaway entries**
    - **Snipe protection**
    - **Premium badge**
    - **Priority support**
    - **Custom command**
    - **Custom Last.fm embed**
    - **AI command access**
    - **Social notifications** (Instagram, TikTok, YouTube, Twitch)
  </Accordion>

  <Accordion title="Custom Last.fm embed">
    Replace the default now-playing card with your own [embed](/resources/scripting/embeds) or [container](/resources/scripting/containers) template. Also unlocks `,lastfm customcommand` (`cc`).
    See the full variable list on the [Last.fm integration page](/integrations/lastfm#custom-now-playing-embed-personal-premium).
    <CodeGroup>
    ```javascript Syntax
    ,lastfm mode (embed)
    ,lastfm cc (trigger)
    ```
    ```javascript Example
    ,lastfm mode {embed}$v{title: {track.name}}$v{description: {artist.name} • {proper(track.plays)} plays}$v{thumbnail: {album.cover}}
    ,lastfm cc .
    ```
    </CodeGroup>
  </Accordion>

  <Accordion title="Self prefix">
    Set your own command prefix across all servers.
    <CodeGroup>
    ```javascript Syntax
    ,selfprefix (prefix)
    ```
    ```javascript Example
    ,selfprefix !
    ```
    </CodeGroup>
  </Accordion>

  <Accordion title="Self aliases">
    Create personal command aliases that work in any server.
    <CodeGroup>
    ```javascript Syntax
    ,selfalias add (alias) (command)
    ```
    ```javascript Example
    ,selfalias add b balance
    ```
    </CodeGroup>
  </Accordion>

  <Accordion title="Grok AI">
    Chat with Grok-powered AI via `,grok`.
    <CodeGroup>
    ```javascript Syntax
    ,grok (question)
    ```
    ```javascript Example
    ,grok explain quantum computing simply
    ```
    </CodeGroup>
  </Accordion>

  <Accordion title="Voice transcription">
    Transcribe voice messages with `,transcribe`.
    <CodeGroup>
    ```javascript Syntax
    ,transcribe (reply to voice message)
    ```
    </CodeGroup>
  </Accordion>

  <Accordion title="Background removal">
    Remove the background from an image.
    <CodeGroup>
    ```javascript Syntax
    ,transparent (image URL)
    ```
    ```javascript Example
    ,transparent https://adore.rest/example.png
    ```
    </CodeGroup>
  </Accordion>
</AccordionGroup>
