## Overview

adore includes AI-powered commands for answering questions, chatting, and more. `,ask` is **free**. Grok, voice transcription, and background removal require [Personal Premium](/overview/donator-perks).

## Ask

Get answers from adore's AI assistant. Uses MINIMAX with Gemini fallback.

<CodeGroup>

```javascript Syntax
,ask (question)
```

```javascript Example
,ask what is the capital of France?
```

</CodeGroup>

You can also reply to a voice message to have adore transcribe and answer it.

## Grok

Chat with Grok-powered AI.

<Card title="Subscription Required" icon="medal" iconType="duotone" horizontal>
  You must have [Personal Premium](/overview/donator-perks) to use `,grok`.
</Card>

<CodeGroup>

```javascript Syntax
,grok (question)
```

```javascript Example
,grok explain how Discord bots work
```

</CodeGroup>

## Voice transcription

Transcribe voice messages to text.

<Card title="Subscription Required" icon="medal" iconType="duotone" horizontal>
  You must have [Personal Premium](/overview/donator-perks) to use `,transcribe`.
</Card>

Reply to a voice message with:

```javascript
,transcribe
```

Or use `,ask` / `,grok` while replying to a voice message to transcribe and respond in one step.

## Background removal

Remove the background from an image with `,transparent`.

<Card title="Subscription Required" icon="medal" iconType="duotone" horizontal>
  You must have [Personal Premium](/overview/donator-perks) to use `,transparent`.
</Card>

<CodeGroup>

```javascript Syntax
,transparent (user/url/attachment)
```

```javascript Example
,transparent https://adore.rest/example.png
```

</CodeGroup>
