# adore docs images

Screenshots for the documentation go here. Use a **dark Discord theme** when capturing.

## Structure

```
images/
├── security/          # antinuke, moderation, fake perms
├── configuration/     # tickets, starboard, voicemaster, etc.
├── integrations/      # spotify, lastfm setup
└── overview/          # customization, premium
```

## Guidelines

- Capture from a server running **adore** (not other bots)
- Use PNG for static screenshots, GIF for short demos
- Reference in MDX with `<Frame><img src="/images/..." /></Frame>`
- Keep filenames descriptive: `antinuke-overview.png`, `ticket-panel-create.png`

The previous bleed.bot screenshots have been removed. Add adore-branded screenshots as you capture them.
