## Overview

Social notifications post to a Discord channel when a creator uploads content or goes live. Each platform is its own command group. Feeds are checked about every **5 minutes**.

<Info>
  Social notifications are a [Personal Premium](/overview/donator-perks) perk. Setup still requires `Manage Server`.
</Info>

## Shared subcommands

Most platforms share the same pattern:

| Subcommand | Purpose |
| ---------- | ------- |
| `add (channel) (target)` | Start notifying in a channel |
| `remove (channel) (target)` | Stop notifying |
| `list` | Show all subscriptions |
| `message (target) (message) --pingable (on\|off)` | Custom embed / text |
| `message view (target)` | Preview the template |

`--pingable on` allows role and user mentions in the notification message.

## YouTube

Dispatched when a channel uploads a video or short (RSS feed).

```javascript
,youtube never gonna give you up
,youtube add #videos https://youtube.com/@MrBeast
,youtube remove #videos @MrBeast
,youtube list
,youtube message @MrBeast {embed}$v{title: {youtube.video.title}} --pingable on
,youtube message view @MrBeast
```

Default template example:

```text
{embed}$v{message: **{youtube.channel.name}** uploaded a __**New Video**__!}$v{author: {youtube.channel.name} && {youtube.channel.icon}}$v{title: {youtube.video.title}}$v{url: {youtube.video.url}}$v{color: #FF0000}$v{image: {youtube.video.thumbnail_url}}$v{button: {youtube.video.url} && Video && /e && enabled}$v{button: {youtube.channel.url} && Channel && /e && enabled}
```

## TikTok

```javascript
,tiktok add #clips charlidamelio
,tiktok remove #clips charlidamelio
,tiktok list
,tiktok live charlidamelio on
,tiktok message charlidamelio New TikTok! --pingable on
,tiktok message view charlidamelio
```

## Instagram

```javascript
,instagram xpllrne
,instagram add #feed shaq
,instagram remove #feed shaq
,instagram list
,instagram story shaq
,instagram stories shaq on
,instagram highlight shaq
,instagram message shaq New post! --pingable on
,instagram message view shaq
```

- `,instagram (username)` — view a profile
- `story` — view active stories
- `stories (username) on|off` — toggle story notifications (requires an existing `add` subscription)
- `highlight` — view highlight reels (no autopost)

## How it works

| Platform | Free source | Interval |
| -------- | ----------- | -------- |
| YouTube | Public Atom / RSS feed | ~5 min |
| TikTok | Existing scraper / TikWM | ~5 min |
| Instagram | Existing Instagram service | ~5 min |
| Twitter | FxTwitter | ~5 min |
| Subreddit | Reddit `.json` | ~5 min |
| SoundCloud | Public page scrape | ~5 min |
| Pinterest | Public board scrape | ~5 min |
| Twitch | Public page live check | ~5 min |
| Kick | Public Kick API | ~5 min |

No paid third-party APIs are required.

See all template variables under [Variables → Notifications](/resources/scripting/variables#youtube).
