title: Commands
description: Prefix commands for configuring adore's ticket system.
icon: terminal
---

<Callout icon="terminal" color="#6d84a0">
  The command group is `ticket`. The alias `tickets` also works. Default prefix is `,`.
</Callout>

Setup commands generally need **Manage Server**. In-ticket actions (`close`, `claim`, etc.) use ticket staff permissions.

---

## Overview

| Command | Description |
| ------- | ----------- |
| `,ticket` | Same as `,ticket overview` |
| `,ticket overview` | Dashboard: panels, forms, stats, transcript channel |
| `,ticket stats [member]` | Server ticket stats, or stats for a staff member |
| `,ticket list` | List open tickets |

---

## Panels

| Command | Description |
| ------- | ----------- |
| `,ticket panel` | Open the interactive panel manager |
| `,ticket panel create (name)` | Create a panel |
| `,ticket panel delete (name)` | Delete a panel and its options |
| `,ticket panel send (name) [channel]` | Deploy panel to a channel |
| `,ticket resend (panel) [channel]` | Redeploy / resend a panel message |
| `,ticket panel mode (name) [button\|dropdown]` | Set or view display mode |
| `,ticket panel category (name) [category\|remove]` | Set or view ticket category |

---

## Options

| Command | Description |
| ------- | ----------- |
| `,ticket option` | Interactive option manager |
| `,ticket option (panel)` | Manage options for a panel |
| `,ticket option create (panel) (label)` | Add a new option |
| `,ticket option delete (option_id)` | Remove an option by ID |

---

## Forms

| Command | Description |
| ------- | ----------- |
| `,ticket form` | List all forms |
| `,ticket form create (name) (title)` | Create a form |
| `,ticket form delete (name)` | Delete a form |
| `,ticket form field (name)` | List fields on a form |
| `,ticket form field (name) add (label) [long]` | Add a field |
| `,ticket form field (name) delete (field_key)` | Remove a field |

---

## Server configuration

| Command | Description |
| ------- | ----------- |
| `,ticket transcript [channel\|remove]` | Set transcript destination, or generate one in a ticket |
| `,ticket staff [role\|remove]` | Global staff role for ticket permissions |
| `,ticket blacklist [member\|role]` | Toggle (or list) the ticket blacklist |

There is **no** `,ticket log`; transcripts use the transcript channel.

---

## In-ticket actions

| Command | Description |
| ------- | ----------- |
| `,ticket close [channel] [reason]` | Close a ticket |
| `,ticket reopen [channel] [reason]` | Reopen a ticket |
| `,ticket claim [channel] [reason]` | Claim a ticket |
| `,ticket unclaim [channel]` | Remove the claimer |
| `,ticket rename [channel] (name)` | Rename the ticket channel |
| `,ticket delete [channel] [reason]` | Delete the ticket channel (sends transcript first) |
| `,ticket move [channel] (option\|category) [reason]` | Move to another option, or into a category |
| `,ticket reason (claim\|close\|reopen\|delete\|move) [channel] (reason)` | Update a stored action reason |
| `,ticket allow (member\|role)` | Allow someone to see the ticket |
| `,ticket allow list` | List explicit allows |
| `,ticket deny (member\|role)` | Deny someone from seeing the ticket |
| `,ticket trainee` | Trainee override help |
| `,ticket trainee grant (member\|role)` | Grant speak/claim/close on this ticket |
| `,ticket trainee revoke (member\|role)` | Revoke trainee overrides |
| `,ticket trainee list` | List trainee overrides |

---

## Not included

Personal/server **ticket profiles** (`ticket profile` / `ticket profiles`) are not implemented.
