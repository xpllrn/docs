Forms let you ask questions in a Discord modal before adore creates the ticket channel. Responses are posted in the new ticket as a summary embed.

## Limits

- **25** forms per server
- **5** fields per form (Discord modal limit)
- Form title max **45 characters** (Discord limit)
- Form name max **50 characters**

---

## Create a form

```javascript
,ticket form create BugReport "Bug Report"
```

- `name` - internal identifier used in commands
- `title` - shown at the top of the modal members see

List existing forms:

```javascript
,ticket form
```

Delete a form:

```javascript
,ticket form delete BugReport
```

---

## Add fields

After creating a form, the bot prompts you to add fields:

```javascript
,ticket form field BugReport add "What happened?"
,ticket form field BugReport add "Steps to reproduce" long
```

```javascript
,ticket form field BugReport delete steps_to_reproduce
```

List fields on a form:

```javascript
,ticket form field BugReport
```

Each field becomes one question in the modal. You can add up to **5** fields per form.

Field types:

| Type | Behaviour |
| ---- | --------- |
| Short (default) | Single-line answer |
| Long | Paragraph answer |

---

## Attach a form to an option

Forms are linked **per option**, not per panel.

1. Run `,ticket panel`
2. Select your panel → **Options**
3. Select the option → **Form**
4. Pick the form from the dropdown (or **No Form** to remove)

When a member clicks that option:

1. If the form has fields → modal opens with your questions
2. If the form has no fields → ticket opens immediately (no modal)
3. After submit → ticket channel is created and responses are summarized in the greeting

---

## Example workflow

```javascript
,ticket form create Report "User Report"
```

```javascript
,ticket form field Report add "Who are you reporting?"
,ticket form field Report add "What rule did they break?"
,ticket form field Report add "Evidence (links)"
```

Then attach **Report** to your "Report User" option in the panel manager and redeploy:

```javascript
,ticket panel send Support #support
```

---

## Tips

- Keep labels short - Discord limits modal field labels to 45 characters.
- Put the most important question first.
- Forms are reusable: one "Application" form can be attached to multiple options on different panels.
- Deleting a form does not delete options that referenced it - re-link them in the option manager.
