## Overview

adore's default prefix is `,`. You can change it per server or set a personal prefix that follows you across servers.

## Server prefix

<CodeGroup>

```javascript Syntax
,prefix (symbol)
,prefix remove
```

```javascript Example
,prefix !
,prefix .
,prefix remove
```

</CodeGroup>

- Requires **administrator** permission (or owner/fake owner)
- `,prefix remove` resets to the default `,`
- Most commands also work as **slash commands** regardless of prefix

## Personal prefix

Set a prefix that works across all servers you use adore in:

```javascript
,selfprefix !
,selfprefix remove
```

- Prefix must be **3 characters or fewer** with no spaces
- Your self prefix overrides the server prefix for you only
