## Overview

Ping on join (`,poj`) sends a brief ping when a new member joins, then deletes the message — a "ghost ping" that notifies the channel without leaving a visible ping.

## Commands

<CodeGroup>

```javascript Syntax
,poj add (channel)
,poj remove (channel)
,poj list
```

```javascript Example
,poj add #welcome
,poj add #general
,poj remove #welcome
```

</CodeGroup>

## Limits

| Tier | Max channels |
| ---- | ------------ |
| Free | 25 |
| Premium | 50 |
